# 方案文件夹 — 设计文档

**日期**: 2026-05-21
**状态**: 已确认

## 概述

在方案中心和工作台添加可嵌套的树形方案文件夹，让用户可以将方案按层级结构分类管理。

## 数据模型

### 文件夹 (Folder)

存储在新文件 `data/folders.json`，与 `solutions.json` 同级。

```json
{
  "id": "folder_xxx",
  "name": "双11活动",
  "parentId": null,
  "createdAt": "2026-05-21T...",
  "updatedAt": "2026-05-21T..."
}
```

字段说明：
- `id`: 唯一标识，格式 `folder_<uuid hex>`
- `name`: 文件夹名称
- `parentId`: 父文件夹 id，`null` 表示根级文件夹
- `createdAt` / `updatedAt`: 时间戳

### 方案 — 新增字段

在现有 Solution 对象上增加一个字段：

- `folderId`: string | null — 所属文件夹 id。`null` 表示"未分类"。

一个方案只属于一个文件夹。

## 后端 API

### 新增 FolderStore 类 (`folder_store.py`)

- `list_folders()` → 返回按树形结构组织的全部文件夹
- `create_folder(name, parent_id)` → 新建文件夹
- `update_folder(id, name)` → 重命名
- `delete_folder(id)` → 删除文件夹，将其下方案 folderId 置为 null
- `move_folder(id, parent_id)` → 移动文件夹到新父级

### 新增 API 路由

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/folders` | 获取文件夹树 |
| POST | `/api/folders` | 新建文件夹 |
| PUT | `/api/folders/:id` | 重命名文件夹 |
| DELETE | `/api/folders/:id` | 删除文件夹 |
| PUT | `/api/folders/:id/move` | 移动文件夹 |

### 方案 API 修改

- `POST /api/solutions/drafts` — 支持 `folderId` 字段
- `PUT /api/solutions/:id` — 支持 `folderId` 字段
- `GET /api/solutions` — 支持 `?folderId=xxx` 查询参数筛选
- 新增 `PUT /api/solutions/:id/move` — 移动方案到目标文件夹

### 删除文件夹时的行为

删除文件夹时，该文件夹下所有方案（直接归属的）的 `folderId` 设为 `null`，即归入"未分类"。子文件夹递归删除。

## 前端变更

### 新增组件: FolderTree.vue

递归树形组件，功能：
- 渲染文件夹树，支持展开/折叠
- 新建文件夹（根级或子文件夹）
- 右键菜单：重命名、新建子文件夹、删除
- 拖拽文件夹调整层级
- 点击文件夹筛选方案列表
- "未分类" 作为特殊的虚拟根节点

### 方案中心 (SolutionCenter.vue) 变更

左侧栏改造：
- 顶部保留"方案中心"标题和新建立草稿按钮
- 新增 FolderTree 组件
- 状态筛选和搜索保持在文件夹树下方
- 方案列表联动：选中文件夹后只显示该文件夹下的方案
- "全部"标签显示所有方案（无视文件夹）
- 搜索结果也显示方案所属文件夹
- 方案右键菜单：移动到、复制、删除
- 支持拖拽方案到文件夹上实现移动

### 工作台 (NormalMode.vue) 变更

"已发布方案"面板改造：
- 顶部保留"已发布方案"标题和切换按钮
- 新增 FolderTree（只读模式，只展示包含已发布方案的文件夹）
- 方案列表联动

### 新建草稿时选择文件夹

新建草稿时，如果当前选中了一个文件夹，新方案默认归入该文件夹。提供下拉选择器让用户可以选择目标文件夹。

## 交互细节

### 文件夹操作
- **新建**: 点击"+ 新建文件夹"，输入名称后创建。如果选中了文件夹则在其中创建子文件夹。
- **重命名**: 右键 → 重命名，内联编辑或弹窗。
- **删除**: 右键 → 删除，弹窗确认。方案归入未分类。
- **拖拽移动**: 文件夹可拖到另一个文件夹上变为其子文件夹。拖到空白区域变为根级。

### 方案跨文件夹移动
- 拖拽方案到目标文件夹
- 或右键方案 → "移动到..." → 弹出文件夹选择器

### 未分类
- 没有归属文件夹的方案自动显示在"未分类"下
- "未分类"始终显示在文件夹树底部
- 不可删除、不可重命名

## 迁移策略

现有所有方案的 `folderId` 字段默认为 `null`，自动归入"未分类"。无需数据迁移脚本。

## 实施范围

| 模块 | 工作内容 |
|------|----------|
| 后端 | FolderStore 类 + 6 个 API + 方案 API 修改 |
| 方案中心 | FolderTree 组件 + 左侧栏改造 + 拖拽/右键 |
| 工作台 | 已发布方案面板改造 + FolderTree |
| 现有数据 | 无需迁移，folderId 为 null 即未分类 |

## 不包含的内容

- 不实现批量操作（批量移动方案等）
- 不实现文件夹颜色/图标自定义
- 不实现回收站/恢复已删除文件夹
- 不实现文件夹级别的权限控制
