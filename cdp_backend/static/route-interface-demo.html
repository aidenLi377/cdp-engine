路由是什么 定义

路由负责把“请求地址 + 请求方法”分发给正确的处理函数。它强调的是 请求怎么被转发。

@app.route("/api/solutions") def list_solutions(): return jsonify(solution_store.list_solutions())
接口是什么 定义

接口强调的是系统对外提供了什么能力。比如 GET /api/solutions 这个能力，就是“获取方案列表”。

接口 = URL + 方法 + 参数 + 返回结果 示例: GET /api/solutions 返回: [{ "id": "...", "name": "..." }]
