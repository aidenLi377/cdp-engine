打标签后按标签筛选。最轻量，只需在现有方案数据中加一个标签字段。

B

单层文件夹

左侧文件夹列表 + 右侧方案列表。类似"分类"的概念，每个方案归属一个分类。

C

可嵌套文件夹

树形目录结构，支持子文件夹。最灵活但 UI 和数据结构都更复杂。
