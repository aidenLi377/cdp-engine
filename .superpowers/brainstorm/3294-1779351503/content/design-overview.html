设计方案总览

基于你的选择：可嵌套树形文件夹、左侧栏内嵌、全功能操作

1. 数据模型

新增文件夹实体，方案增加 folderId 字段。一个方案只属于一个文件夹，folderId 为空时属于"未分类"。

📁 文件夹 (Folder)
{
  id: "folder_xxx",
  name: "双11活动",
  parentId: null,  // null=根目录
  createdAt: "...",
  updatedAt: "..."
}
📄 方案 — 新增字段
{
  ...现有字段,
  folderId: "folder_xxx",  // 新增
  ...
}

2. 后端 API（新增）

GET /api/folders 获取全部文件夹（返回树形结构）
POST /api/folders 新建文件夹 { name, parentId? }
PUT /api/folders/:id 重命名文件夹 { name }
DELETE /api/folders/:id 删除文件夹（方案自动归入未分类）
PUT /api/folders/:id/move 移动文件夹 { parentId }
PUT /api/solutions/:id/move 方案移动到文件夹 { folderId }

5. 交互细节

新建方案时选文件夹

新建草稿时弹出选择器或默认放入当前选中的文件夹。也可创建后移动到其他文件夹。

拖拽移动

方案可以拖到文件夹上实现移动；文件夹也可以拖拽调整父子关系。

右键菜单

方案右键：移动到 / 复制 / 删除。文件夹右键：重命名 / 新建子文件夹 / 删除。

6. 实施范围

后端 FolderStore 类 + 6 个新 API + 方案 API 增加 folderId 支持
方案中心 左侧栏加 FolderTree + 方案列表联动 + 拖拽/右键菜单
工作台 已发布方案面板加入 FolderTree + 筛选联动
迁移 现有方案 folderId 为 null → 自动归入"未分类"
