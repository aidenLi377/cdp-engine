方案文件夹 — 设计文档

2026-05-21 · 状态：已确认

概述

在方案中心和工作台添加可嵌套的树形方案文件夹，让用户可以将方案按层级结构分类管理。

1. 数据模型

📁 文件夹 (Folder) — 新文件 data/folders.json
{
  id: "folder_xxx",
  name: "双11活动",
  parentId: null,    // null=根级
  createdAt: "...",
  updatedAt: "..."
}
📄 方案 — 新增一个字段
{
  ...现有字段不变,
  folderId: "folder_xxx",
  ...
}

null = "未分类"

2. 后端 API

新增 FolderStore + 文件夹 API
GET   /api/folders   获取文件夹树
POST   /api/folders   新建文件夹
PUT   /api/folders/:id   重命名
DELETE /api/folders/:id   删除
PUT   /api/folders/:id/move   移动层级
方案 API 修改
创建/更新草稿支持 folderId
列表查询支持 ?folderId=xxx
新增 PUT /api/solutions/:id/move
删除文件夹 → 方案自动归入"未分类"
子文件夹递归删除

5. 交互细节

文件夹操作

新建、重命名、删除、拖拽调整层级。右键菜单触发。

方案操作

拖拽或右键菜单移动到目标文件夹。新建时选择归属文件夹。

未分类

folderId=null 的方案自动归入。不可删除不可重命名。
