# 参数辅助助手 插件

## 安装方式

1. 打开 Chrome 扩展页：`chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择目录 `chrome-extension/databank-automation`

## 当前支持域名

- 项目页：`http://127.0.0.1:5173`
- 达摩盘页：`https://databank.tmall.com/*`

## 功能说明

项目页点击“自动化圈人”后，插件会自动新建一个后台参数页面，并执行固定三步自动化：

1. 点击“参数粘贴”
2. 写入当前 JSON
3. 点击“确定”

默认采用半静默模式：

- 成功时在后台标签页完成，不主动切走当前页面
- 失败时自动切到目标标签页，方便人工查看卡住的位置

执行前会先做页面状态校验，包括：

- 当前是否为目标参数页面
- 参数粘贴入口是否存在且可交互
- 当前是否已有未关闭的导入弹窗

## 本地联调步骤

1. 加载插件
2. 启动前端：`http://127.0.0.1:5173`
3. 登录达摩盘页面
4. 在项目页点击“自动化圈人”验证链路

当前版本不需要手动填写插件 ID。只要插件已加载，并且项目页与达摩盘页都在当前 Chrome 中打开，就可以直接通信。

## 打包分发

如需把当前插件源码打包发给同事，可直接压缩整个目录：

`chrome-extension/databank-automation`

同事收到压缩包后，解压并按“安装方式”中的步骤通过开发者模式加载即可。
